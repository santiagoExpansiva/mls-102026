Widget Select Dog for Adoption

Este widget permite selecionar um cachorro para adoção de uma lista, com filtros por raça e idade. Suporta navegação por teclado (setas para cima/baixo, Enter para selecionar).

Pontos Fortes: Interface intuitiva, filtros funcionais, acessibilidade com teclado e ARIA.

Restrições: Requer lista de cachorros no formato correto; filtros são textuais simples.
